# Healthcare Claims Analysis Project

## Project Overview
This project demonstrates **end-to-end data analytics** on healthcare claims data, including data cleaning, transformation, validation, and interactive visualizations. It highlights skills in **Python, ETL, and Power BI** and is designed to showcase insights for healthcare providers and insurers.

## Key Features
- Automated **ETL pipeline** to process raw claims data (`etl_claims.py`).  
- **Data analysis scripts** to generate insights (`analysis_claims.py`).  
- **Validation scripts** to ensure data quality and integrity (`validate_claims.py`).  
- Interactive **Power BI dashboard** visualizing:
  - Total claims by insurance type
  - Provider and patient statistics
  - Claim trends over time
- Cleaned and structured data for advanced analysis.  
- Exported dashboard in Power BI for reporting and presentations.
