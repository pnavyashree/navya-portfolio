
## Tools & Technologies
- **Python** – Data cleaning, transformation, and ETL pipeline
- **Pandas & NumPy** – Data manipulation and aggregation
- **Power BI** – Interactive visualizations and dashboards
- **Git & GitHub** – Version control and portfolio hosting

## Key Features
- Cleaned and transformed raw claims data for analysis
- Created an ETL script to automate the data processing pipeline
- Built interactive Power BI dashboards for insights on:
  - Total claims by insurance type
  - Provider and patient statistics
  - Claim trends over time
- Exported dashboards to PDF for sharing and presentation

   ```bash
